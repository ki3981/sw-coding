## 📁 폴더 구조

| 폴더 | 내용 |
|------|------|
| `1장 파이썬 소개` | 파이썬 기초 소개 |
| `2장 변수와 연산자` | 변수, 자료형, 연산자 |
| `3,4장 조건문 반복문` | if문, for/while문 |
| `5장 함수 입출력` | 함수 정의 및 입출력 |
| `6장 리스트` | 리스트 자료구조 |
| `7장 딕셔너리` | 딕셔너리 자료구조 |
| `8장` | 모듈과 활용 |
| `중간고사 전 과제` | 3,4장 / 5장 과제 |
| `중간고사 이후 과제` | 8장 과제 (`8장 과제.ipynb`) |
| `시험대비 문제 풀이` | 시험 대비 연습 문제 |
| `작년 시험문제` | 기출문제 모음 |
